Hi,

Your pvpgn server is running on gateway machine? If so, then, have you redirected 6118 port of your external IP  on server to 192.168.0.15:6112 in ipnat?
for ex:
rdr interface external_ip/32 port 6118 -> 192.168.0.15 port 6112 tcp
rdr interface external_ip/32 port 6118 -> 192.168.0.15 port 6112 udp

And I think the configuration line in address_translation.conf should be the next:
192.168.0.15:6112    external_IP:6118    192.168.0.0/24 ANY
